# Online Tour Booking Website

## Overview

A responsive Tour Booking Website developed using HTML, CSS, JavaScript and Bootstrap. The website allows users to browse tour packages, view destinations, register, login and book tours through an interactive user interface.

## Features

- User Registration
- User Login
- Tour Package Listing
- Destination Management
- Booking System
- Responsive Design
- Navigation Menu
- Browser History Management

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap
- JSON Server

## Project Structure

- Home Page
- Registration Page
- Login Page
- Tour Booking Page
- Add Places Module
- Navigation Component
- Contact Us Page

## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Start JSON Server

```bash
npx json-server db.json
```

4. Open index.html in browser
